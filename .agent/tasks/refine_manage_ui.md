# 작업: Manage UI 고도화

## 완료 항목
- **오버레이 스타일 (Overlay Styles)**: Minimal, Clean, Bold 스타일 추가 및 적용.
- **키워드/태그 통합 (Keywords/Tags Consolidation)**: Keywords 탭 제거, Tags 탭으로 통합.
- **불필요한 탭 제거**: 중복 Keywords 탭 제거.

## 잔여 작업 (Remaining Tasks)
1. **"Style" 탭 고도화**: Profile, LoRA, Character, Model, Embedding 등 5개 하위 탭 UI 일관성 개선.
2. **전반적인 UI/UX 오디트**:
    - 마이크로 애니메이션 또는 호버 효과 강화.
    - 모든 탭의 반응형 레이아웃 최적화.
3. **최종 검증**: 전체 애플리케이션 실행 및 스튜디오 페이지 연동 확인.

## V3 관련 신규 항목
- **Multi-Character UI**: `scene_character_actions` 테이블 기반 장면별 캐릭터 액션 편집 UI.
- **Scene Builder UI**: `scene_tags` 테이블 기반 배경/시간/날씨 태그 선택 UI.
- **Storyboard List**: `GET /storyboards` 기반 저장된 스토리보드 목록/불러오기 UI.
