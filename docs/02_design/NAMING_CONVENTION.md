# 서비스 네이밍 컨벤션

**상태**: Active (v1.0)
**최종 업데이트**: 2026-03-17
**관련 문서**: `CLAUDE.md` 용어 정의 섹션

---

## 1. 언어 정책

**기본 언어: 한국어**. 단, 크리에이터 도구로서 외국어 표현이 자연스러운 부분은 영어를 허용한다.

---

## 2. 영역별 언어 기준

### 영어 허용

| 영역 | 예시 | 이유 |
|------|------|------|
| 글로벌 네비게이션 | Home, Studio, Library, Settings | 크리에이터 도구 관례 |
| Studio 워크스페이스 탭 | Script, Stage, Direct, Publish | 워크플로우 단계명 |
| Library 서브탭 | Characters, Styles, Voices, Music, LoRAs | 에셋 카테고리 |
| 기술 기능명 버튼 | Auto Run, Render, TTS, BGM | 기술 용어로 자연스러움 |
| SD 파라미터 라벨 | CFG Scale, CLIP Skip, Steps, Sampler | 전문 파라미터 |
| Backend API 에러 | "Storyboard not found" | 개발자 대상 응답 |

### 한국어 필수

| 영역 | 예시 | 이유 |
|------|------|------|
| 도메인 용어 | 채널, 시리즈, 영상, 씬, 캐릭터, 화풍 | 사용자 이해도 |
| 토스트/알림 메시지 | "저장 완료", "생성에 실패했습니다" | 사용자 소통 |
| CRUD 버튼 | 새 영상, 삭제, 저장, 수정 | 행동 유도 |
| 상태 표시 | 생성 중, 완료, 실패, 대기 중 | 사용자가 읽는 값 |
| 칸반 컬럼 | 초안, 제작 중, 렌더 완료, 게시됨 | 워크플로우 상태 |
| 에러 화면 | "문제가 발생했습니다" | 사용자 안내 |
| 빈 상태 메시지 | "씬이 없습니다", "캐릭터가 없습니다" | 사용자 안내 |
| placeholder | "예: 내 애니메이션 채널" | 입력 가이드 |
| 확인 대화상자 | 제목 + 버튼 모두 한국어 | 일관성 |

### 혼합 금지

| 규칙 | 금지 예시 | 올바른 예시 |
|------|----------|------------|
| 한 문장에 한영 혼합 | "Critical: 재생성을 권장합니다" | "심각: 재생성을 권장합니다" |
| 제목 한국어 + 버튼 영어 | 제목: "채널 삭제" / 버튼: "Delete" | 제목: "채널 삭제" / 버튼: "삭제" |
| 같은 화면에서 같은 개념 한영 혼용 | "씬이 없습니다" (StageTab) + "No scenes yet" (ScenesTab) | 둘 다 "씬이 없습니다" |

## 3. 도메인 용어 사전

### 계층 구조

| 코드 (영문) | UI (한국어) | 설명 |
|-------------|------------|------|
| Project | 채널 | 콘텐츠 발행 채널 (최상위) |
| Group | 시리즈 | 같은 화풍의 영상 묶음 |
| Storyboard | 영상 | 개별 쇼츠 영상 단위 |
| Scene | 씬 | 영상 내 개별 장면 |

### Studio 워크플로우

| 코드/UI (영어) | 설명 |
|---------------|------|
| Script | 대본 생성 (AI 채팅) |
| Stage | 사전 준비 (이미지, TTS, BGM) |
| Direct | 씬별 상세 편집 |
| Publish | 렌더링 + 게시 |

### 칸반 상태

| 코드 (영문) | UI (한국어) |
|-------------|------------|
| draft | 초안 |
| in_prod | 제작 중 |
| rendered | 렌더 완료 |
| published | 게시됨 |

### AI 실행 모드

| 코드 (영문) | UI (한국어) | 설명 |
|-------------|------------|------|
| guided (interaction_mode) | 가이드 모드 | 단계별 확인하며 진행 |
| auto (interaction_mode) | 자동 모드 | AI가 전체 자율 실행 |
| FastTrack (skip_stages) | 빠른 생성 | Director/Research/Concept 건너뜀 |

> `Express`, `Quick`, `hands_on`은 **폐기된 용어**. 코드/문서에서 발견 시 위 용어로 교체.

### 콘텐츠 용어

| 코드 (영문) | UI (한국어) |
|-------------|------------|
| Character | 캐릭터 |
| Style Profile | 화풍 |
| Scene Text | 자막 (영상 위 텍스트) |
| Caption | 캡션 (영상 설명글/해시태그) |
| Voice Preset | 음성 프리셋 |
| Music Preset | BGM 프리셋 |
| Background | 배경 |
| LoRA | LoRA (영어 그대로) |

## 4. 토스트 메시지 표준 패턴

### 성공

```
"{도메인 용어} {동작} 완료"
예: "영상 저장 완료", "캐릭터 생성 완료", "시리즈 삭제 완료"
```

### 실패

```
"{도메인 용어} {동작}에 실패했습니다"
예: "영상 저장에 실패했습니다", "이미지 생성에 실패했습니다"
```

### 안내/경고

```
"{조건/상태} 설명문"
예: "먼저 채널을 생성하세요", "씬을 저장한 뒤 렌더하세요"
```

### 금지 패턴

| 금지 | 이유 | 올바른 표현 |
|------|------|------------|
| "Failed to save" | 영어 | "저장에 실패했습니다" |
| "Storyboard auto-saved" | 영어 | "영상 자동 저장 완료" |
| "Create a group to start" | 영어 | "시리즈를 먼저 생성하세요" |
| "Character saved" | 영어 | "캐릭터 저장 완료" |

## 5. 확인 대화상자 표준

```
제목: "{도메인 용어}을(를) 삭제하시겠습니까?"
설명: "이 작업은 되돌릴 수 없습니다."
확인: "삭제"
취소: "취소"
```

---

## 6. 빈 상태 메시지 표준

```
제목: "{도메인 용어}이(가) 없습니다"
설명: "{안내 동작}"
버튼: "{동작}"
```

예시:
```
제목: "씬이 없습니다"
설명: "먼저 스크립트를 생성하세요"
버튼: "스크립트 생성"
```
