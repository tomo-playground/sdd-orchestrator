# Scene Clothing Override

> 상태: 미착수

## 배경

캐릭터 의상은 캐릭터 프로필에 고정. 장면별로 다른 의상(잠옷, 교복, 수영복 등)을 입히려면 캐릭터를 복제해야 함.

## 목표

- 씬 단위로 캐릭터 의상 태그를 오버라이드
- 캐릭터 기본 의상은 유지하되 특정 씬에서만 변경
- 프롬프트 엔진에서 오버라이드 태그 우선 적용

## 범위

| 항목 | 설명 |
|------|------|
| Scene Clothing UI | SceneCard에서 의상 태그 변경 |
| Override Logic | 씬 의상 > 캐릭터 기본 의상 우선순위 |
| Prompt Integration | 12-Layer Builder에서 오버라이드 반영 |
| Reset | 기본 의상으로 되돌리기 |

## 수락 기준

| # | 기준 |
|---|------|
| 1 | 씬별 의상 태그 변경 가능 |
| 2 | 변경하지 않은 씬은 캐릭터 기본 의상 유지 |
| 3 | 프롬프트에 오버라이드 의상 정확히 반영 |
