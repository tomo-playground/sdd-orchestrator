# Visual Tag Browser

> 상태: 미착수

## 배경

현재 태그가 텍스트 목록으로만 표시되어 직관적 이해가 어려움. 태그별 예시 이미지를 표시하면 선택 정확도 향상.

## 목표

- 태그 목록에 예시 이미지 썸네일 표시
- Danbooru 또는 자체 생성 이미지 활용
- 태그 카테고리별 시각적 탐색

## 범위

| 항목 | 설명 |
|------|------|
| Tag Thumbnail | 태그별 대표 이미지 1장 |
| Image Source | Danbooru API 또는 자체 SD 생성 |
| Category View | 카테고리별 그리드 레이아웃 |
| Cache | 썸네일 이미지 로컬 캐싱 |

## 수락 기준

| # | 기준 |
|---|------|
| 1 | 태그 선택 시 예시 이미지 표시 |
| 2 | 이미지 로딩 실패 시 텍스트 fallback |
| 3 | 카테고리별 태그 시각적 탐색 가능 |
